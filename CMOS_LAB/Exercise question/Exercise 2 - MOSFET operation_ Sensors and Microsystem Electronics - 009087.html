Simulation of transistors

Most Spice components also use something called a model. A model is a bundle of common parameters (and equations) that are shared between multiple components. For example, the oxide thickness is common to all transistors so is part of the model, but the geometrical properties (eg. transistor width and length) of each transistor is specified on the component.

In order to run simulations for any circuit containing transistors, you will need to include the following Spice operative (using the button ):

.inc library.mod

This line lets the simulator inherit the properties of PMOS and NMOS transistors we need from the library file. Even without the ".inc library.mod" command, the simulations will run but the results will be for some other generic transistor models provided by LTspice. So, it is important to include the ".inc library.mod". 

Note: Please use transistors PMOS4 and NMOS4 from the component library

Transistor as a resistor

The triode region of a transistor can be approximated by a straight I-V curve through the origin. This is the same characteristic as a resistor! In other words, when you can keep the I-V characteristic within this region, the transistor will behave like a resistor.

Sweep V2 from 0 to 3.3V (the maximum allowed voltage for this process) with a step of 0.1 V and plot the current through NMOS M1 and resistor R1.

Transistor sizes are not as accurate as you might think. Process variations can quickly lead to variation on transistor currents of +-10%. It is therefore not a good idea to try and design a transistor for a specific current with 0.01um accuracy on the width or length.

The current through a transistor varies with vGS−vTH. So we can use the gate-voltage to effectively make a voltage-controlled current source, or also called a transconductance amplifier.

Sometimes the dependency on the threshold voltage is quite annoying! If the temperature changes, then the threshold voltage vTH will also change, causing vGS−vTH to change and thus the ids-current through the transistor!

In order to avoid this issue, a current mirror can be used. Build the following circuit:

         

M1 (W/L=1u/1u) and M2 (W/L=1u/1u) form the current mirror. Don't forget to specify the supply voltage of 3.3 V through V2.

Sweep V1 from 0 to 3.3V and plot the current through I2 and M2. Try changing a few sizes and check the current versus V1 (vds).

  1. How does this circuit work? Explain in your own words how you can prove that the current is copied to M1.
  2. How do the W/L ratio of the transistors influence the output drain current characteristic?
  3. How do the lengths of the transistors influence the current characteristic (keeping W/L constant)?
Your answer:

1. Since the gate of M1 is connected with the gate of M2,  so Vgs1= Vgs2, 

that means M1 and M2 have the same inversion level,  and ids are proportional to W/L, Since M1 and M2 have the same width and length, that's why ids(M1)= ids (M2)

2. when W/L increases, the output drain current of the transistor will increase. 

 3.  when we increase the length of the transistor, the output conductance will decrease,  in the saturation region,  the current will be nearly equal to constant, (Vds has low effect on ids, the current slop nearly equal to 0)  if we decrease the length, the current value will be affected by the change of Vds (go increase).  the slop of current will increase (Ids - Vds curve).

The MOS switch is the easiest way to think about a transistor. We typically distinguish two states the transistor can be in:

  • If vgs > vt, then the transistor is on, the drain and source are connected by a channel under the gate.
  • If vgs < vt, then the transistor is off, the drain and source are not connected by a channel under the gate.

Of course, the I-V characteristic is still respected. If vds is small, the transistor will operate in the triode region, and if vds > vgs - vt then the transistor will be in saturation. For a switch to operate properly we want the transistor to act like a short-circuit. In other words, we want high current and low voltage drops.

Build the following circuit with W/L for M1 as 0.4u/0.35u:

    

The input should be a pulsed voltage source V1 between 0 and 3.3V, delay, rise and fall time are 1ns, pulse width is 1us and period is 2us.

  1. The NMOS is not strong enough to make the voltage go to 0V. Choose the transistor M1 sizes such that the output voltage drops below 0.1V while keeping the used chip area to a minimum.
  2. Copy the obtained netlist file below.
