The goal is to slightly shift the timing of a rising edge of a signal (image below is only an example, use timing constraints from the text). The reference input signal has a pulse width of at least 75ns. When the signal is high, the circuit should wait for about 40ns before pulling the output high. When the input goes low, the output goes down as well almost immediately.

The input signal can potentially stay high for extended periods of time. The minimum amount of time between two pulses is at least 20ns. You can use current-starving techniques to create slow slopes for this.

Additionally, you can implement the following features:

  1. Make the delay controllable by an additional analog voltage
    • Make the delay as linearly depending on the analog input as possible 
  2. Delay the falling edge by the same amount as the rising edge 
    • Verify that this also works when the delay is bigger than 80% of the pulse width
